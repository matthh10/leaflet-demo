# How to make maps with {leaflet} in R

Maps are often useful in public health to understand the spatial distribution of cases and identify clusters. The {leaflet} package in R is useful for quickly creating interactive maps using free, open-source data [@leaflet].

This repo hosts the code for a tutorial which can be accessed here: [Leaflet in R demo](https://matthh10.github.io/leaflet-demo/tutorial.html)
